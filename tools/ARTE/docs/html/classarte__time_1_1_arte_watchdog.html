Detailed Description

A watchdog timer for threads or any long running task.

Note
ArteWatchdog implements threading.Timer. An instance of ArteWatchdog will instantiate a threading.Timer in the constructor and a new threading.Timer every reset.
    timeout (unsigned int): The timeout in seconds.
