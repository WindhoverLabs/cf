ARTE Usage

ARTE requires one command line argument for a JSON configuration file. An example is included below:
{
  "name": "ARTE test config file",
  "version": 0.1,
  "date": "2017-09-22T00:00:00.000Z",
  "timeout": 300,
  "timeouts": 5,
  "watchdog": 5,
  "majorframe": 200,
  "clients": [
      {
      "name": "Pyliner",
      "command": "python2 challenge.py",
      "output": "test_script1.txt",
      "cwd": "../../tools/pyliner/"
    },
    {
      "name": "Airliner",
      "command": "./core-bin -M TEST",
      "output": "test_script2.txt",
      "cwd": "../../build/softsim/default/target/exe/"
    }
  ]
}

  • timeout The overall timeout for the test.
  • timeouts A generic timeout for anything that would otherwise pend forever.
  • watchdog A watchdog timeout for threads.
  • majorframe The number of minor frames in a major frame.
  • clients The clients that should connect to ARTE server.
    • name The name of the client.
    • command The command to execute a client.
    • output The file to pipe output to.
    • cwd The directory to execute the command in.
